# Text Area

The `TextArea` component is a multi-line text input field. Like the `Input` component, it is designed to be used within a `TextField` composition to provide accessible labels, descriptions, and error messages.

<ComponentExample src="svelte/text-area/TextAreaUsage.svelte">
  <TextAreaUsage client:load />
</ComponentExample>

## Examples

### Rows

Use the `rows` property to control the initial height of the text area. The default is `3`.

<ComponentExample src="svelte/text-area/TextAreaRows.svelte">
  <TextAreaRows client:load />
</ComponentExample>

### Validation (Invalid State)

When the `TextField` or `TextArea` is marked as `invalid`, the `FieldError` message will be displayed and the text area will show an error state.

<ComponentExample src="svelte/text-area/TextAreaInvalid.svelte">
  <TextAreaInvalid client:load />
</ComponentExample>

### Disabled State

Setting the `disabled` state on the `TextField` automatically disables the `TextArea` via context.
